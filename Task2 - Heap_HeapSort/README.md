## Quick Start
- User can change or add the priority and jobs they need into the jobs list as (prioirty, job name) in the following section inside the program

```shell
# Example data: (priority, job_name)           
jobs = [
    (2, "Job A"),
    (5, "Job B"),
    (3, "Job C"),
    (1, "Job D")
]
```
